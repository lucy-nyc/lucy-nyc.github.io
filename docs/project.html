01 · The problem

Re-reading notes isn't studying. Testing yourself is.

The most effective way to study is active recall — quizzing yourself instead of passively re-reading. But writing your own practice questions is slow, and generic quiz generators tend to invent facts that were never in your material.

Test Maker fixes both: it writes the questions for you in seconds, and it constrains every single one to be answerable from the exact text you paste — so a practice test never quizzes you on something your professor never covered.

03 · The details

Built for a real study session

✓
Three question formats
Multiple choice, true/false, and short answer — combined in any mix you choose.
✓
Strictly grounded generation
Questions are constrained to the pasted material via structured JSON output, so the test never invents facts.
✓
Client-side PDF import
Reads text out of an uploaded PDF in the browser with pdf.js, straight into the material box.
✓
Answer keys with explanations
Reveal or hide answers as you go, each with a short why-this-is-right note.
✓
Print-ready PDF export
Generate a clean blank test for practice, or a full answer key, with jsPDF.
