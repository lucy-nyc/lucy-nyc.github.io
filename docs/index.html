Study tool

Turn your notes into a mock test

Paste in class material — notes, a chapter, a study guide — and Test Maker writes you a practice exam grounded strictly in what you gave it.

Try it live → View on GitHub

What it does

Three question types

Multiple choice, true/false, and short answer — pick any mix.

You set the shape

Choose how many questions and the difficulty level.

Grounded in your material

Every question is answerable from the text you paste — no made-up facts.

Import a PDF

Pull text straight out of a PDF into the material box.

Answer keys

Show or hide answers with explanations as you study.

Export to PDF

Download a blank test or a full answer key to print.
